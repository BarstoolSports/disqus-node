add

Adds an entry to the whitelist.

Signature:

Disqus#whitelists.add(options[, cb])

Usage:

// Node-style
disqus.whitelists.add({
    forum: '',
    api_secret: 'asdfghkj', // can be set globally
    access_token: '12345678', // can be set globally
    // defaults shown
    notes: '',
    email: [],
    user: []
}, function (err, result) {...})

// Promise-style
disqus.whitelists.add({
    forum: '',
    // defaults shown
    notes: '',
    email: [],
    user: []
})
